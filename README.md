## ✨ Core Design & Functionality Features

* **🎨 Midnight & Cream Visual Palette:** Warm, elegant, high-contrast dark theme designed to minimize eye strain while keeping notes highly readable.
* **⚡ Dynamic Note Rendering:** Instantly updates your board workspace with real-time word counting, active note tracking badges, and timestamp formatting.
* **🏷️ Fast-Drafting Canvas:** Jot down instant thoughts immediately using the prominent top-level utility drafting card.
* **📌 Interactive Notes Grid:** Responsive card layout featuring:
  * Dynamic timestamp logging (`Jul 15, 04:42am`).
  * Fast note deletion and edit transitions.
  * Pinning utilities (top-right card indicators).
  * Unique decorative design details mimicking traditional card binding holes.
* **🔍 Real-Time Fuzzy Search:** Filter through dozens of written notes dynamically inside the header search bar on the fly.

## 🛠 Tech Stack

* **Structure:** Semantic HTML5 Structure
* **Styling:** Custom CSS3 Variables, Fluid Flexbox/CSS Grid layouts, FontAwesome 6.4 Icon Core
* **Engine:** Vanilla JavaScript (Reactive DOM updates, local state handling, search filtration regex)
